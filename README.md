# Fruitify
This simple fruits website to demo Tailwind CSS capabilities

## Pages 
1. Home Page
    - Hero section
    - Featured fruit cards
    - testimonials
    - call to action buttons

2. Shop/Catalogue (products page)
    - Multi-column layout
    - sidebar filters
    - sale badges - add to card

3. Product detail
    - large galleries
    - quantity selectors
    - related items

4. Cart/checkout
    - order summary details
    - Input fields
    - Sticky checkout buttons

5. Responsive Navbar and Footer
